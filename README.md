## 1. Быстрая установка и запуск проекта

В папке с проектом содержится файла `docker-compose.yml` для развертывания проекта в Docker контейнерах.

```cmd
git clone https://github.com/Alexcey-ko/Python-homework-11
cd Python-homework-11
docker compose up --wait
```

После развертывания контейнеров, FastApi должно быть доступно по [ссылке](http://localhost:8000/docs).

## 2. Изменения проекта по сравнению с предыдущим заданием №10

В проекте были выполнены следующие изменения:

- изменена структура таблицы `Sbook`: добавлено новое поле `sbookid`; которое стало единственным ключевым полем;
- добавлены новые методы в сервисах и репозиториях для реализации логики роутеров: получение списка бронирований, полчение бронирования по `sbookid`, удаление бронирования по `sbookid`.
- сущности `entities` заменены на `Pydantic` схемы `schemas`;
- удалены модули с консольным приложением: `client`;
- добавлены роутеры: `routers`.
